# Customer Segmentation using PCA and K-Means Clustering

## Project Overview

This project demonstrates Customer Segmentation using Unsupervised Machine Learning techniques. The objective is to identify hidden customer groups based on demographic and service-related information. Principal Component Analysis (PCA) is used for dimensionality reduction, followed by K-Means Clustering to segment customers into meaningful groups.


## Dataset

The dataset contains customer information from a telecom company, including:

- Gender
- Senior Citizen
- Partner
- Dependents
- Tenure
- Phone Service
- Internet Service
- Online Security
- Online Backup
- Device Protection
- Tech Support
- Streaming TV
- Streaming Movies
- Contract
- Paperless Billing
- Payment Method
- Monthly Charges
- Total Charges

## Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Handle Missing Values
5. Remove Duplicate Records
6. Encode Categorical Features
7. Feature Scaling using StandardScaler
8. Apply PCA for Dimensionality Reduction
9. Determine Optimal Clusters using the Elbow Method
10. Validate Clusters using Silhouette Score
11. Apply K-Means Clustering
12. Analyze Customer Clusters
13. Generate Business Personas
14. Export Final Clustered Dataset
